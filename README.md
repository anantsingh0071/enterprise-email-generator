# Enterprise Email Generator

An AI-powered Enterprise Insurance Email Generator using Google's Gemini model.

## Features

- Dataset-driven email generation
- Chain-of-Prompt architecture
- Enterprise email review
- EML file generation
- Modular Python architecture

## Tech Stack

- Python
- Google Gemini
- python-dotenv

## Project Structure

- Dataset Service
- LLM Service
- Email Generator
- EML Generator
